# Class Layouts — pointer

The authoritative source of class offsets is now
[`hero-layout.md`](hero-layout.md) for the Hero specifically and
**`classes.json`** for everything else (9,125 HOBJ/HSTRUCT classes,
generated by [`../tools/hlbc_parse.py`](../tools/hlbc_parse.py); not
checked in because of size).

Regenerate any time with:

```pwsh
python tools\hlbc_parse.py "D:\SteamLibrary\steamapps\common\Farever\hlboot.dat"
```

Single-class inspection:

```pwsh
python tools\hlbc_parse.py "D:\SteamLibrary\steamapps\common\Farever\hlboot.dat" ent.Hero
```
